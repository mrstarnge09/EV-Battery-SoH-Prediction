## Project Title
**Predicting Electric Vehicle Battery State of Health (SoH) Using Machine Learning**

---

## Problem Statement
Battery degradation is one of the biggest challenges in the Electric Vehicle industry. Predicting the **State of Health (SoH)** helps EV companies:
- Reduce warranty costs
- Improve predictive maintenance
- Enhance customer trust and safety
- Optimize battery performance

This project focuses on predicting battery health using real-world driving and charging parameters.

---

## Dataset Used
- **Synthetic EV Battery Health Dataset**
- Features: Total distance, average speed, ambient temperature, trip duration, charging cycles, fast charging ratio, battery temperature, etc.

---

## Technologies Used
- **Programming**: Python
- **Libraries**: Pandas, NumPy, Scikit-learn, XGBoost
- **Visualization**: Matplotlib, Seaborn
- **Deployment**: Streamlit (Streamlit Cloud)
- **Model**: XGBoost Regressor (Best Performing)

---

## Project Workflow
1. Exploratory Data Analysis (EDA)
2. Feature Engineering
3. Model Training (Linear Regression, Random Forest, XGBoost)
4. Model Evaluation (MAE, RMSE, R² Score)
5. Feature Importance Analysis
6. Web App Development & Deployment

---

## Results
- **Best Model**: XGBoost
- Interactive web application successfully deployed
- Real-time battery health prediction based on user inputs
