# EV Battery State of Health (SoH) Prediction

## Project Title
**Predicting Electric Vehicle Battery State of Health (SoH) Using Machine Learning**

## Problem Statement
Electric Vehicle batteries degrade over time due to repeated charging cycles, temperature variations, and usage patterns. Predicting the **State of Health (SoH)** of EV batteries is crucial for:
- Improving battery lifespan
- Predictive maintenance
- Enhancing customer experience and safety
- Helping EV companies like Ola Electric, Ather Energy, and Tata Motors reduce warranty costs

This project aims to build a machine learning model that can accurately predict the remaining health of an EV battery.

## Dataset Used
- **Dataset Name**: Synthetic EV Battery Health Dataset
- **Source**: Kaggle
- **Link**: [https://www.kaggle.com/datasets/prajwalmpoojary/synthetic-ev-battery-health-data](https://www.kaggle.com/datasets/prajwalmpoojary/synthetic-ev-battery-health-data)
- **Description**: Contains battery parameters such as voltage, current, temperature, charging cycles, and battery health percentage.
